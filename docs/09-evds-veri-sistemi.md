# EVDS - Elektronik Veri Dağıtım Sistemi

## Purpose
TCMB'nin ekonomik verileri kamuoyuyla paylaştığı EVDS sistemini tanıtmak.

## EVDS Nedir?
EVDS (Elektronik Veri Dağıtım Sistemi), TCMB'nin faiz oranları, döviz kurları, enflasyon, parasal büyüklükler gibi ekonomik verilere dinamik, etkileşimli ve zaman serisi biçiminde erişim sağlayan çevrimiçi platformudur.

## Kullanım Alanları
- Araştırmacılar ve akademisyenler için tarihsel veri analizi
- Yazılım geliştiriciler için API üzerinden programatik veri erişimi
- Bankacılık ve finans sektörü için güncel piyasa verisi takibi

## Erişim
EVDS'e tcmb.gov.tr üzerinden web arayüzüyle veya bir API anahtarı alınarak programatik şekilde erişilebilir.

## Bu Projeyle İlişkisi
Bu RAG asistanı şu an EVDS'in canlı verilerine bağlı değildir; burada anlatılan bilgiler statik referans belgeleridir. İleride EVDS API'si entegre edilerek asistanın güncel faiz/kur verilerini gerçek zamanlı sorgulaması sağlanabilir.
